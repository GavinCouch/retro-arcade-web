# Retro Arcade Web

A browser-based emulator hub (starting with NES) where users **upload their own ROM files locally** and play in the browser.

## Features (NES)
- BYO-ROM upload (`.nes`)
- Keyboard controls (WASD + J/K + Enter + Space)
- Audio support (Safari-compatible)
- Reset (cold boot)
- Save States (localStorage + autosave)

## Legal / ROM Policy
This project **does not provide or host ROM files**.
Only use ROMs you legally own or have permission to use.
